Trabalho desenvolvido para a disciplina **Modelos de Regressão** da Universidade Federal Fluminense.
A base de dados é composta por **33.600 filmes** (1960–2024) obtida via [Base dos Dados](https://basedosdados.org),
e a variável resposta é `rating_imdb` — a nota média atribuída pelos usuários do IMDb.
